# Kelron (Qwen 2.5-14B) 온프라미스 구축 권장 사양서

고객사 서버실에 Kelron을 구축하기 위한 하드웨어 사양 가이드입니다.
**14B 모델 + RAG(문서 검색) + 다중 사용자 접속** 환경을 기준으로 산정되었습니다.

## 1. GPU 서버 사양 (가장 중요)

모델의 응답 속도와 동시 처리량을 결정합니다. **VRAM(비디오 메모리)** 용량이 핵심입니다.

| 등급 | 추천 GPU 구성 | VRAM 합계 | 특징 및 용도 | 예상 비용 |
| :--- | :--- | :--- | :--- | :--- |
| **High-End (권장)** | **NVIDIA A100 (80GB) x 1** | 80GB | **[고성능/대규모]**<br>- 속도 매우 빠름<br>- 수십 명 동시 접속 가능<br>- 긴 문서 처리에 유리 | 수천만 원대 (서버 포함) |
| **Standard (타협)** | **NVIDIA L40S (48GB) x 1** | 48GB | **[가성비/기업용]**<br>- A100의 합리적 대안<br>- 추론(Inference) 전용으로 설계됨 | 1~2천만 원대 |
| **Minimum (최소)** | **RTX 4090 (24GB) x 2** | 48GB | **[소규모/저비용]**<br>- 소비자용 GPU 2장 연결<br>- 24GB 1장으로는 부족함 (OOM 위험)<br>- 5~10명 소규모 팀 | 수백만 원대 |

> **⚠️ 주의 (RTX 3090/4090 1장 불가)**
> 14B 모델은 기본 약 28GB VRAM을 차지합니다. 24GB 카드 1장에는 **원본 화질(FP16)**로 올릴 수 없습니다. 화질을 낮추는 양자화(Quantization)를 해야 겨우 올라가므로, 기업용 납품 시에는 **최소 48GB 이상 (2장 연결)**을 권장합니다.

## 2. 기타 하드웨어 사양

| 항목 | 최소 사양 | 권장 사양 | 비고 |
| :--- | :--- | :--- | :--- |
| **CPU** | Intel Xeon Silver 16 Core↑ | Intel Xeon Gold 24 Core↑ | 데이터 전처리 및 RAG 검색 엔진(Vector DB) 구동용 |
| **RAM** | 64 GB | **128 GB** | PDF 파싱 및 대량의 텍스트 처리를 위해 넉넉해야 함 |
| **Storage** | SSD 1TB | **NVMe SSD 2TB** | RAG용 문서 데이터와 벡터 인덱스 저장 공간 필요 |
| **OS** | Ubuntu 22.04 LTS | Ubuntu 22.04 LTS | Windows Server보다 Linux 환경이 AI 호환성 압도적 |

## 3. 네트워크 및 보안
- **폐쇄망 설치 가능**: Docker Image 파일로 반입 (`docker load`)
- **포트 오픈**: 
    - 8000 (API 서버)
    - 8501 (데모 UI)
    - 6333 (Vector DB)
